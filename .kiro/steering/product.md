# Product Overview

最終更新: 2025-12-14

Docker ベースの MCP (Model Context Protocol) サーバーを安全かつ簡便に運用するための Web コンソール。Bitwarden 連携で秘密情報をディスクに残さず注入し、コンテナ操作・カタログからの導入・機能検査までをブラウザから完結させる。

## Core Capabilities

- **セキュアなシークレット注入**: Bitwarden CLI と連携し、資格情報をローカル保存せずに安全に供給。
- **OAuth 対応カタログ接続**: PKCE/state 付き認可 URL を生成し、取得したアクセストークンを Fernet で暗号化保存（キーは環境変数で差し替え）。
- **GitHub トークン管理**: Bitwarden から GitHub Personal Access Token (PAT) を取得・暗号化して保存し、API レート制限を回避してカタログやリポジトリにアクセス。
- **コンテナライフサイクル管理**: MCP サーバーコンテナの起動・停止・再起動、状態確認、ログ取得を UI で実行。
- **MCP インスペクション**: ツール/リソース/プロンプトのエクスポーズ内容をリアルタイムに確認し、挙動を把握。
- **カタログ導入**: GitHub MCP レジストリ (contents API) を既定ソースとし、フェイルオーバーとキャッシュで安定取得。ワンクリックで MCP サーバーを追加・更新。
- **署名検証ポリシー**: 署名検証の強制/監査モードをポリシーで切り替え、許可リストやメトリクス記録に対応。
- **外部/E2B ゲートウェイ**: 許可リスト検証と定期ヘルスチェックを備えた外部ゲートウェイ登録・監視を提供。
- **安全なセッション運用**: セッションタイムアウトと CORS 設定により開発/チーム利用を前提とした安全性を担保し、リソース制限付きゲートウェイコンテナをセッション単位で起動。

## Target Use Cases

- ローカルで複数の MCP サーバーを試験・運用したい開発者。
- OAuth が必要なサーバーや外部/E2B ゲートウェイを安全に繋ぎたいチーム。
- 署名検証やメトリクス監査を求められるセキュリティ要件下で GUI から導入・検証したいユースケース。
- プライベートリポジトリにある MCP サーバーを GitHub トークン連携でスムーズに利用したいケース。

## Value Proposition

Docker と Bitwarden の煩雑な CLI 操作を抽象化し、視覚的な管理体験でセットアップから検証までを短時間で完了できる。OAuth トークン暗号化・署名検証ポリシー・ゲートウェイ許可リストに加え、GitHub トークン連携によるレート制限の緩和（匿名制限を軽減するが、PATスコープやアカウント種別に依存）も備え、秘密情報を残さずコンプライアンス要件に沿った導入が可能。個人開発からチーム運用まで安心して利用できる。
